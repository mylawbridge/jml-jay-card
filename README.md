## How to update

- **Booking link** — search for `BOOKING LINK` comment in `index.html` and replace the URL
- **Headshot** — re-embed a new photo as base64 or update the `src` attribute in the `<img>` tag
- **Contact info** — search for phone/email/website values in `index.html`

## Deploy
This site is deployed via GitHub Pages from the `main` branch.
